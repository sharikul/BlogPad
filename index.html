BlogPad is an upcoming blogging engine complete with its own templating engine that is designed to make it easy for almost anyone to get started with a blog on their own server.

Getting started

Though not a CMS, BlogPad is intended to be used 'as one'. So you'll need to delegate routing to BlogPad through a .htaccess file.

RewriteEngine On
RewriteBase /blog/

RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule . /blog/index.php [L]

blog denotes that BlogPad's files are stored within a directory called /blog/. If it's stored in a different directory, blog will be replaced with the name of the directory on your computer/server. If it's not in a directory (i.e. root), RewriteBase will be RewriteBase / and RewriteRule will be RewriteRule ./index.php [L].
